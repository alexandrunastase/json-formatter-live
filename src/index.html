The motivation behind the app

I created this app in order to have a quick way to format code without needing to open another tab, in neither the browser nor the IDE. I think that PWA is a good fit for this, because a stand-alone desktop app is overkill for a JSON formatter, and a web app is not as easy to access, since you need an open browse first and it would add yet another tab.

Principles

  • Ease of use
  • Open (both analytics and source code)
  • No ads or intrusive tracking
  • Decent looking design
